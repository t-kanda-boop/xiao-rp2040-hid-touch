# XIAO RP2040 Android Touch (HID Digitizer)

This project turns Seeed Studio XIAO RP2040 into a USB touch screen device for Android.

## Features

- Absolute coordinate touch
- No scaling required
- No Android app required
- HID Digitizer implementation
- TinyUSB + Pico SDK

## Build

```bash
mkdir build
cd build
cmake ..
make
